# FLASSY STUDIO

Visual AI content studio website — local development.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS v4
- Motion
- next-themes (dark / light)

## Run locally

```bash
npm install
npm run dev
```

Open [http://127.0.0.1:3000](http://127.0.0.1:3000).

If the file watcher hits system limits (rare on large machines), use a production serve instead:

```bash
npm run build && npm start
```

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint
